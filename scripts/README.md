# Scripts

## Automation Scripts

- `sync-all.sh` - Sync all repositories
- `sync-repo.sh` - Sync single repository
- `setup.sh` - Initial setup script
- `schedule-sync.sh` - Schedule automatic sync

## Usage

```bash
# Sync all
bash scripts/sync-all.sh

# Sync one
bash scripts/sync-repo.sh <repo-name>
```
